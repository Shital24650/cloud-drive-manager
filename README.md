# Cloud Drive Manager

Cloud Drive Manager is a web application built with Angular that allows users to upload, preview, download, rename, and manage files in the cloud.

## Features

- File upload
- File preview (images and PDFs)
- Download files
- Rename files
- Delete files
- Search files
- Responsive UI
- Authentication support
- Shareable file links

## Tech Stack

- Angular
- Angular Material
- Supabase (Cloud Storage & Auth)

## Run Locally

Install dependencies:

npm install

Start development server:

ng serve

Open:

http://localhost:4200

## Build Project

ng build

Build files will be generated inside:

dist/
